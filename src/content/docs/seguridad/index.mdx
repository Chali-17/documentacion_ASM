## Introducción

El **Módulo de Seguridad** del Sistema ASM Dashboard es el centro de control para la administración de usuarios, permisos, accesos y políticas de seguridad del sistema. Este módulo permite a los administradores gestionar de forma centralizada todos los aspectos relacionados con la seguridad de la plataforma.

### Características Principales

- Gestión completa de usuarios y roles
- Control granular de permisos por módulos y vistas
- Monitoreo de sesiones activas en tiempo real
- Auditoría completa de todas las acciones del sistema
- Autenticación de dos factores (2FA)
- Políticas de seguridad configurables

### Requisitos de Acceso

Para acceder al módulo de seguridad, el usuario debe:
- Tener una cuenta activa en el sistema
- Contar con el rol de **Administrador** o **Seguridad**
- Tener los permisos necesarios asignados
